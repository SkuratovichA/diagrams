@startuml_seq
@actor Administrator
== Create some Audit settings ==
Administrator -> Watcher @actmsgstart create new Audit Template (i.e. Audit settings : goal, scope, ...) @actmsgend
Watcher -> Watcher @actmsgstart save Audit Template in database @actmsgend
Administrator <-- Watcher @actmsgstart Audit Template UUID @actmsgend
== Launch a new Audit ==
Administrator -> Watcher @actmsgstart launch new Audit of the Openstack infrastructure resources\nwith a previously created Audit Template @actmsgend
Administrator <-- Watcher @actmsgstart Audit UUID @actmsgend
Administrator -> Watcher @actmsgstart get the Audit state @actmsgend
Administrator <-- Watcher @actmsgstart ONGOING @actmsgend
Watcher -> Watcher @actmsgstart compute a solution to achieve optimization goal @actmsgend
Administrator -> Watcher @actmsgstart get the Audit state @actmsgend
Administrator <-- Watcher @actmsgstart SUCCEEDED @actmsgend
== Get the result of the Audit ==
Administrator -> Watcher @actmsgstart get Action Plan @actmsgend
Administrator <-- Watcher @actmsgstart recommended Action Plan and estimated efficacy @actmsgend
Administrator -> Administrator @actmsgstart verify the recommended actions\nand evaluate the estimated gain vs aggressiveness of the solution @actmsgend
== Launch the recommended Action Plan ==
Administrator -> Watcher @actmsgstart launch the Action Plan @actmsgend
Administrator <-- Watcher @actmsgstart Action Plan has been launched @actmsgend
Watcher -> Watcher @actmsgstart trigger Actions on Openstack services @actmsgend
Administrator -> Watcher @actmsgstart get the Action Plan state @actmsgend
Administrator <-- Watcher @actmsgstart ONGOING @actmsgend
Administrator -> Watcher @actmsgstart get the Action Plan state @actmsgend
Administrator <-- Watcher @actmsgstart SUCCEEDED @actmsgend
@enduml_seq
